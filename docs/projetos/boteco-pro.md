# Boteco Pro

## Descrição

Sistema completo de gestão para bares e restaurantes com autenticação Firebase, banco de dados e interface Flutter moderna.

## Tecnologias Utilizadas

- Flutter (Dart)
- Firebase (Autenticação e Banco de Dados)

## Repositório

[GitHub: BotecoPro](https://github.com/Monynha-Softwares/BotecoPro)

## Funcionalidades

- Gestão de pedidos
- Autenticação de usuários
- Interface moderna para bares e restaurantes
