## Setup & local development

### Prerequisites

- Flutter SDK (3.x+)
- Dart SDK (2.19+)
- Convex account / project setup

### Quick start

1. Clone the repository

```bash
git clone https://github.com/Monynha-Softwares/Boteco-Pro.git
cd boteco-pro
```

1. Install dependencies

```bash
flutter pub get
```
