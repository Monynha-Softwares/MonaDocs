# TypeScript

## Descrição

TypeScript é uma linguagem de programação que é um superconjunto de JavaScript, adicionando tipagem estática opcional.

## Por que usamos

- Melhor manutenção de código
- Detecção de erros em tempo de desenvolvimento
- Melhor suporte a IDEs

## Projetos que utilizam

- Monynha-com
- Vários outros projetos web
