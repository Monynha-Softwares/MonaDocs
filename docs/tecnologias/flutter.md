# Flutter

## Descrição

Flutter é um framework de UI do Google para criar aplicações nativas compiladas para mobile, web e desktop a partir de uma única base de código.

## Por que usamos

- Desenvolvimento cross-platform
- Interface nativa
- Hot reload para desenvolvimento rápido
