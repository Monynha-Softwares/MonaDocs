# Coolify

## Descrição

Coolify é uma plataforma de deployment self-hosted que permite fazer deploy de aplicações web, bancos de dados e serviços em servidores próprios. Ele suporta múltiplas linguagens e frameworks, oferecendo uma interface amigável para gerenciamento de infraestrutura.

## Por que usamos

- Controle total sobre a infraestrutura
- Deploy automatizado de aplicações
- Suporte para múltiplas stacks tecnológicas
- Interface web intuitiva para gerenciamento
- Alternativa open-source ao Heroku/Vercel

## Como usar

### Instalação

Coolify pode ser instalado em um servidor Ubuntu/Debian:

```bash
curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash
```

### Deploy básico

1. Conecte seu repositório Git
2. Configure as variáveis de ambiente
3. Escolha o tipo de aplicação (Node.js, PHP, etc.)
4. Faça deploy automático
